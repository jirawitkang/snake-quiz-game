นักเรียนใช้ Room Code ที่ได้จากคุณเพื่อเข้ามา Join

เข้าห้อง (สำหรับนักเรียน)

นักเรียนกรอกรหัสห้องที่ครูให้ แล้วใส่ชื่อของตัวเองเพื่อต่อเข้าห้อง

รายชื่อผู้เล่นในห้อง:

    * สถานะช่อง / แต้มทอย / การตอบคำถามจะถูกอัปเดตแบบ realtime

    เมื่อกดเริ่มรอบใหม่ นักเรียนทุกคนจะสามารถทอยลูกเต๋าได้รอบละ 1 ครั้ง
    (Step ถัดไปจะเพิ่มช่วงคำถาม + ให้เดินหน้า/ถอยหลังตามผลตอบ)
