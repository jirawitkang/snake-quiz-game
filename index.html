Snake Quiz – Multiplayer

สร้างห้อง (สำหรับครู / Host)

ตั้งค่าเกมก่อนสร้างห้อง

เมื่อกด "ยืนยันสร้างห้อง" แล้ว จะสร้าง Room Code และสามารถให้ผู้เล่น Join ได้

ขั้นตอน: 1) กรอกชื่อ Host → 2) ตั้งค่าเกม → 3) ยืนยันสร้างห้อง แล้วแชร์ Room Code ให้นักเรียน

เข้าห้อง (สำหรับนักเรียน)

นักเรียนกรอกรหัสห้องที่ครูให้ แล้วใส่ชื่อของตัวเองเพื่อต่อเข้าห้อง

รายชื่อผู้เล่นในห้อง:

    * สถานะผู้เล่น / ทอยล่าสุด / ตอบล่าสุด จะอัปเดตแบบ realtime

    1) กด "เริ่มรอบใหม่" → นักเรียนทอยลูกเต๋ารอบละ 1 ครั้ง
    2) เมื่อทุกคนทอยครบ กด "เริ่มคำถาม" → ขึ้นนับถอยหลัง 3,2,1 และเข้าสู่ช่วงตอบ
    3) เมื่อหมดเวลา หรือทุกคนตอบครบ กด "เฉลยคำถาม & ขยับตัว"
